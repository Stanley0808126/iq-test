Interactive IQ Test - 图形推理测试
To deploy: upload this folder to any static hosting
- index.html (26KB, standalone, no dependencies)

Deployment options:
1. GitHub Pages: create repo, push files, enable Pages
2. Vercel: `npx vercel --cwd=iqtest --yes`
3. Netlify: drag-drop iqtest folder
4. Serve locally: `cd iqtest && python -m http.server 3000`
